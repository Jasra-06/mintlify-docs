## 8 Topics to Cover

<AccordionGroup>
  <Accordion title="1 — EdTech" icon="graduation-cap">
    **EdTech** = using digital tools and platforms to enhance or enable learning.

    - Online courses, live classes, auto-assessments, digital certificates
    - Market projected to surpass **\$600 billion by 2027**
    - AI-powered platforms are the fastest-growing segment
  </Accordion>

  <Accordion title="2 — LMS (Learning Management System)" icon="desktop">
    An **LMS** creates, delivers, manages, and tracks online learning.

    | Feature | What It Does |
    | --- | --- |
    | Course Builder | Create modules with video, PDFs, quizzes |
    | Progress Tracking | Monitor learner completion and engagement |
    | Assessments | Auto-graded quizzes and exams |
    | Certifications | Digital certs on course completion |
    | Analytics | Insights into learner performance |
  </Accordion>

  <Accordion title="3 — SaaS" icon="cloud">
    **SaaS** = cloud-based software accessed via subscription. No installation needed, always updated.

    - Recurring monthly/annual revenue model
    - **TeachGrid is a SaaS product** — institutions subscribe, not build
  </Accordion>

  <Accordion title="4 & 5 — HashInkludes & TeachGrid" icon="building">
    - HashInkludes = innovation-driven EdTech startup
    - TeachGrid = AI-powered unified learning platform at [teachgrid.io](https://teachgrid.io/)
    - Replaces 5–10\+ disconnected tools with one system

    <Card title="Full Company Profile →" icon="building" href="/get-started/about">
      Read the About page for the full picture.
    </Card>
  </Accordion>

  <Accordion title="6 — The Problem TeachGrid Solves" icon="wrench">
    Most institutions juggle Zoom \+ Google Classroom \+ Excel \+ WhatsApp \+ separate cert tools = chaos.

    | Pain Point | TeachGrid Solution |
    | --- | --- |
    | Scattered tools | One unified dashboard |
    | Manual certificates | Automated digital certs |
    | No analytics | AI-powered insights |
    | High tool costs | Single subscription |
  </Accordion>

  <Accordion title="7 — Ideal Customer Profiles (ICP)" icon="target">
    - 🏫 **Training Academies** — batches, live courses, certs (50–5,000 learners)
    - 🏢 **Corporate L&D** — employee training at scale
    - 🎓 **Universities** — hybrid and online academic programs
    - 🚀 **EdTech Startups** — white-label LMS infrastructure
  </Accordion>

  <Accordion title="8 — Competitor Research" icon="magnifying-glass">
    Research these key players and understand their strengths vs. TeachGrid:

    - **Moodle** — open-source, complex, university-focused
    - **Canvas LMS** — popular in higher education
    - **TalentLMS** — corporate SMB, simple UI
    - **Teachable / Thinkific** — creator-focused course platforms
    - **Google Classroom** — free, basic, limited LMS features

    <Tip>
      Pick 3–5 and build a comparison table — it makes a great presentation slide.
    </Tip>
  </Accordion>
</AccordionGroup>
